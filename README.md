Minha jornada com Excel e Dados

Quando comecei a estudar tecnologia, achava que tudo se resumia ao Excel.
Com o tempo, percebi que o Excel não trabalha sozinho, ele se complementa com outras ferramentas e cria uma base extremamente poderosa para análise de dados.

Neste repositório, compartilho parte da minha evolução nos estudos e projetos utilizando Excel.

1. Excel: onde tudo começou

O Excel foi minha porta de entrada para o universo de dados.

Habilidades que mais fortaleceram minha base:
PROCV, SE, SOMASE e outras funções essenciais
Tabelas Dinâmicas
Limpeza e tratamento de dados
Construção de dashboards
Organização e visualização de informações

Essas habilidades me deram uma visão clara de como os dados se comportam , algo essencial para quem deseja atuar como Analista de Dados.
